# YLproxy
## 项目描述  YLproxy 是一款运行在 Windows 平台上的桌面 GUI 应用程序（基于 .NET / WPF），它的核心职能是：**将需要用户名密码认证的远程 HTTP 代理，转换为本地不需要认证的 HTTP 代理端口**，供模拟器、游戏客户端、浏览器扩展等软件使用。  典型场景：   模拟器/软件 → 127.0.0.1:9001（无需认证）                   ↓             YLproxy 本地代理转换                   ↓         第三方 HTTP 认证代理（需认证）                   ↓                目标网站
