## 当前存储方案（v0.2.0）

### 存储位置

```
YLproxy/
├── data/
│   ├── config.json          ← 代理配置文件
│   └── README.md
├── AppSettings.json         ← 全局配置
└── logs/
    └── ylproxy.log          ← 应用日志
```

### 存储方式

- **格式**：JSON
- **编码**：UTF-8
- **更新**：每次修改后立即保存
- **备份**：可手动备份

### 优点

- ✅ 简单，无需数据库
- ✅ 易于备份和迁移
- ✅ 易于阅读和编辑
- ✅ 占用空间小

### 缺点

- ❌ 不支持并发写入
- ❌ 查询功能有限
- ❌ 大数据量时性能下降
- ❌ 不支持事务

### 序列化/反序列化

```csharp
using System.Text.Json;

public class ProxyDataService
{
    private readonly string _configPath = "data/config.json";
    
    // 读取
    public async Task<List<ProxyItem>> LoadProxiesAsync()
    {
        if (!File.Exists(_configPath))
        {
            return new List<ProxyItem>();
        }
        
        var json = await File.ReadAllTextAsync(_configPath);
        var doc = JsonDocument.Parse(json);
        var proxiesArray = doc.RootElement.GetProperty("proxies");
        
        var proxies = new List<ProxyItem>();
        foreach (var element in proxiesArray.EnumerateArray())
        {
            proxies.Add(JsonSerializer.Deserialize<ProxyItem>(element.GetRawText()));
        }
        
        return proxies;
    }
    
    // 保存
    public async Task SaveProxiesAsync(List<ProxyItem> proxies)
    {
        var config = new
        {
            version = "1.0",
            lastBackupTime = DateTime.UtcNow.ToString("O"),
            proxies = proxies
        };
        
        var json = JsonSerializer.Serialize(config, new JsonSerializerOptions { WriteIndented = true });
        await File.WriteAllTextAsync(_configPath, json, Encoding.UTF8);
    }
}
```

## SQLite 迁移规划（v0.3.0+）

### 迁移目标

| 功能 | JSON | SQLite |
|---|---|---|
| 存储容量 | <10MB | 无限制 |
| 并发读写 | ❌ | ✅ |
| 查询性能 | 低 | 高 |
| 事务支持 | ❌ | ✅ |
| 索引支持 | ❌ | ✅ |

### 实施时间

- **v0.3.0**：引入 SQLite，JSON 兼容迁移
- **v0.4.0**：完全迁移，淘汰 JSON
- **v1.0.0**：仅支持 SQLite

## 数据库表设计

### 表 1：Proxies（代理配置）

```sql
CREATE TABLE Proxies (
    Id INTEGER PRIMARY KEY,
    Name TEXT NOT NULL UNIQUE,
    RemoteHost TEXT NOT NULL,
    RemotePort INTEGER NOT NULL,
    Username TEXT NOT NULL,
    Password TEXT NOT NULL,  -- DPAPI 加密存储
    LocalHost TEXT NOT NULL DEFAULT '127.0.0.1',
    LocalPort INTEGER NOT NULL,
    Status TEXT NOT NULL DEFAULT 'Stopped',  -- Running/Stopped/Failed/...
    ProcessId INTEGER,
    CreatedAt DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
    UpdatedAt DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP
);

-- 创建索引加快查询
CREATE INDEX idx_proxies_status ON Proxies(Status);
CREATE INDEX idx_proxies_local_port ON Proxies(LocalPort);
```

### 表 2：ProxyLogs（代理日志）

```sql
CREATE TABLE ProxyLogs (
    Id INTEGER PRIMARY KEY,
    ProxyId INTEGER NOT NULL,
    LogLevel TEXT NOT NULL,  -- Debug/Info/Warning/Error
    Message TEXT NOT NULL,
    Timestamp DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP,
    FOREIGN KEY (ProxyId) REFERENCES Proxies(Id) ON DELETE CASCADE
);

-- 创建索引加快查询
CREATE INDEX idx_proxy_logs_proxy_id ON ProxyLogs(ProxyId);
CREATE INDEX idx_proxy_logs_timestamp ON ProxyLogs(Timestamp);
```

### 表 3：ProxyStats（代理统计，Phase 13）

```sql
CREATE TABLE ProxyStats (
    Id INTEGER PRIMARY KEY,
    ProxyId INTEGER NOT NULL,
    Date DATE NOT NULL,
    TotalRequests INTEGER DEFAULT 0,
    SuccessRequests INTEGER DEFAULT 0,
    FailedRequests INTEGER DEFAULT 0,
    BytesUpload BIGINT DEFAULT 0,
    BytesDownload BIGINT DEFAULT 0,
    FOREIGN KEY (ProxyId) REFERENCES Proxies(Id) ON DELETE CASCADE,
    UNIQUE(ProxyId, Date)
);
```

### 表 4：Settings（全局设置）

```sql

    Value TEXT NOT NULL,
    UpdatedAt DATETIME NOT NULL DEFAULT CURRENT_TIMESTAMP
```

## 迁移策略

### 迁移流程

```
v0.2.0（JSON）
      │
      ▼
检测到 config.json
      │
      ├─ 第一次启动时创建 SQLite 数据库
      │
      ▼
导入 JSON 数据到 SQLite
      │
      ├─ 逐条读取 JSON 记录
      ├─ 插入 SQLite 数据库
      │
      ▼
验证数据完整性
      │
      ├─ 检查记录数是否相等
      ├─ 检查关键字段是否缺失
      │
      ▼
保留 JSON 备份
      │
      ├─ 复制 config.json 为 config.json.bak
      │
      ▼
切换至 SQLite 模式
      │
      ├─ 后续使用 SQLite 进行读写
      │
      └─ (迁移完成)
```

### 迁移代码示例

```csharp
public class MigrationService
{
    public async Task MigrateFromJsonToSqlite()
    {
        // 1. 检查是否需要迁移
        if (!File.Exists("data/config.json"))
        {
            return;  // 无需迁移
        }
        
        // 2. 备份 JSON
        File.Copy("data/config.json", "data/config.json.bak", overwrite: true);
        
        // 3. 读取 JSON 数据
        var jsonProxies = await _jsonService.LoadProxiesAsync();
        
        // 4. 创建 SQLite 数据库
        await _sqliteService.InitializeDatabaseAsync();
        
        // 5. 导入数据
        foreach (var proxy in jsonProxies)
        {
            await _sqliteService.InsertProxyAsync(proxy);
        }
        
        // 6. 验证
        var importedCount = await _sqliteService.CountProxiesAsync();
        if (importedCount != jsonProxies.Count)
        {
            throw new Exception("迁移失败：记录数不匹配");
        }
        
        // 7. 标记已迁移
        await _sqliteService.SetSettingAsync("Migrated", "true");
        
        Logger.Info("迁移完成");
    }
}
```

```csharp
    // 3. 清除迁移标记
    // ...
    
    Logger.Info("已回滚至 JSON 存储");
}
```
