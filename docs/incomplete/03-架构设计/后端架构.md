### 系统流量流向

```
用户操作
  │
  ├─ 添加代理
  │    → MainViewModel.AddCommand
  │    → AddProxyWindow 弹窗
  │    → AddProxyViewModel 验证输入
  │    → ProxyTester 测试连通性
  │    → PortManager 分配端口
  │    → ProxyDataService 保存 config.json
  │    → 刷新 UI
  │
  ├─ 启动代理
  │    → MainViewModel.StartCommand
  │    → ProxyProcessManager 启动 3proxy
  │    → ConfigGenerator 生成配置文件
  │    → 更新代理状态
  │    → MonitorService 后台监控
  │
  ├─ 删除代理
  │    → MainViewModel.DeleteCommand
  │    → ProxyProcessManager 停止进程
  │    → PortManager 释放端口
  │    → ProxyDataService 删除记录
  │    → 刷新 UI
  │
  └─ 测试代理
       → MainViewModel.TestCommand
       → ProxyTester 测试连通性
       → 输出结果到日志

后台自动
  │
  └─ MonitorService (每 5 秒)
       → ProxyProcessManager.IsRunning()
       → 检测进程存活
       → 更新代理状态
```

### YLproxy.Models（数据模型）

**职责**：定义所有业务实体和数据结构

```csharp
// ProxyItem - 代理数据模型
public class ProxyItem
{
    public int Id { get; set; }
    public string Name { get; set; }
    public string RemoteHost { get; set; }
    public int RemotePort { get; set; }
    public string Username { get; set; }
    public string Password { get; set; }  // (v0.2.0) 明文；(v0.3.0+) DPAPI 加密
    public string LocalHost { get; set; }
    public int LocalPort { get; set; }
    public ProxyStatus Status { get; set; }
    public DateTime CreatedAt { get; set; }
}

// ProxyStatus - 代理状态枚举
public enum ProxyStatus
{
    Created,     // 新建
    Starting,    // 启动中
    Running,     // 运行中
    Stopping,    // 停止中
    Stopped,     // 已停止
    Failed       // 异常
}

// AppConfig - 全局配置
public class AppConfig
{
    public LoggingConfig Logging { get; set; }
    public ProxyConfig Proxy { get; set; }
    public MonitoringConfig Monitoring { get; set; }
}
```

### 2. ProxyTester（代理测试服务）

**职责**：测试代理连通性

```csharp
public interface IProxyTester
{
    Task<ProxyTestResult> TestAsync(ProxyItem proxy);
}

public class ProxyTestResult
{
    public bool Success { get; set; }
    public int DelayMs { get; set; }
    public string ErrorMessage { get; set; }
}

// 实现流程
public async Task<ProxyTestResult> TestAsync(ProxyItem proxy)
{
    var client = new HttpClient(new HttpClientHandler
    {
        Proxy = new WebProxy(
            $"http://{proxy.RemoteHost}:{proxy.RemotePort}",
            false,
            null,
            new NetworkCredential(proxy.Username, proxy.Password)
        )
    });
    
    try
    {
        var sw = Stopwatch.StartNew();
        var response = await client.GetAsync("http://httpbin.org/get");
        sw.Stop();
        
        return new ProxyTestResult
        {
            Success = response.IsSuccessStatusCode,
            DelayMs = (int)sw.ElapsedMilliseconds
        };
    }
    catch (Exception ex)
    {
        return new ProxyTestResult { Success = false, ErrorMessage = ex.Message };
    }
}
```

### 3. ProxyProcessManager（3proxy 进程管理）

**职责**：启动/停止 3proxy.exe

```csharp
public interface IProxyProcessManager
{
    Task<bool> StartAsync(ProxyItem proxy);
    Task<bool> StopAsync(ProxyItem proxy);
    bool IsRunning(ProxyItem proxy);
}

// 启动过程
public async Task<bool> StartAsync(ProxyItem proxy)
{
    // 1. 生成配置文件
    var config = ConfigGenerator.Generate(proxy);
    var cfgPath = $"runtime/3proxy/cfg/{proxy.Id}.cfg";
    File.WriteAllText(cfgPath, config);
    
    // 2. 启动进程
    var process = new Process
    {
        StartInfo = new ProcessStartInfo
        {
            FileName = "runtime/3proxy/3proxy.exe",
            Arguments = cfgPath,
            UseShellExecute = false,
            RedirectStandardOutput = true,
            CreateNoWindow = true
        }
    };
    
    process.Start();
    proxy.ProcessId = process.Id;  // 保存 PID 用于后续管理
    
    await Task.Delay(1000);  // 等待进程启动
    return !process.HasExited;
}
```

### 4. MonitorService（后台监控服务）

**职责**：每 5 秒检测代理进程状态

```csharp
public class MonitorService
{
    private readonly IProxyProcessManager _processManager;
    private readonly Timer _timer;
    
    public void Start()
    {
        _timer = new Timer(CheckStatusAsync, null, TimeSpan.Zero, TimeSpan.FromSeconds(5));
    }
    
    private async void CheckStatusAsync(object state)
    {
        foreach (var proxy in _proxies)
        {
            if (proxy.Status == ProxyStatus.Running)
            {
                bool isRunning = _processManager.IsRunning(proxy);
                
                if (!isRunning)
                {
                    proxy.Status = ProxyStatus.Failed;  // 进程异常
                    OnStatusChanged?.Invoke(proxy);
                }
            }
        }
    }
}
```

## 常见问题

### Q1：如何添加新的服务？

**A：**
1. 在 YLproxy.Models 中定义接口
2. 在 YLproxy.Infrastructure/Core/Proxy 中实现
3. 在 DI 容器中注册
4. 在 GUI 中注入使用

### Q2：为什么要分离这么多项目？

**A：** 
- 便于测试（可单独测试各层）
- 便于维护（职责单一）
- 便于扩展（新功能只影响相关项目）

### Q3：如何处理配置更新？

**A：** 
- AppSettingsService 启动时加载一次
- 运行时若需要重新加载，手动调用 Reload()
- config.json 变更由 ProxyDataService 检测
