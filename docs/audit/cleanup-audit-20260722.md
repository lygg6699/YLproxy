## 二、.gitignore 修改内容

### 修改位置

文件：`.gitignore`（第 34-37 行）

### 修改内容

在「数据和本地配置」部分新增 `AppSettings.json` 忽略规则：

```gitignore
# ====== 数据和本地配置 ======
AppSettings.json
data/config.json
data/config.json.bak
AppSettings.local.json
```

### 修改原因

`AppSettings.json` 是本地配置文件，包含用户特定设置（如日志目录、代理端口范围等），不应提交到仓库。应用首次启动时会自动创建此文件（使用代码默认值），`AppSettings.example.json` 仅作为配置参考模板。
