## MVVM 架构概述

### 三层结构

```
┌─────────────────────────────┐
│        View (XAML)           │   用户界面
│  - MainWindow.xaml          │   展示数据
│  - AddProxyWindow.xaml      │   捕获输入
└────────────────┬────────────┘
                 │ (数据绑定)
                 ▼
┌─────────────────────────────┐
│    ViewModel (C# 代码)       │   业务逻辑
│  - MainViewModel            │   处理命令
│  - AddProxyViewModel        │   状态管理
│  - ViewModelBase (基类)     │
└────────────────┬────────────┘
                 │ (模型操作)
                 ▼
┌─────────────────────────────┐
│      Model (后端服务)         │   数据模型
│  - ProxyItem                 │   业务对象
│  - ProxyDataService          │   数据访问
│  - ProxyTester               │   代理测试
└─────────────────────────────┘
```

### 数据流向

```
用户操作（输入）
      ▼
  View 捕获事件
      ▼
  ViewModel 处理命令
      ▼
  调用后端服务（Model）
      ▼
  Model 返回结果
      ▼
  ViewModel 更新属性
      ▼
  View 通过绑定更新 UI
      ▼
  用户看到变化（输出）
```

## ViewModel 层

### ViewModelBase（基类）

**职责**：提供数据绑定和通知基础

```csharp
public class ViewModelBase : INotifyPropertyChanged
{
    public event PropertyChangedEventHandler PropertyChanged;
    
    protected void OnPropertyChanged([CallerMemberName] string propertyName = null)
    {
        PropertyChanged?.Invoke(this, new PropertyChangedEventArgs(propertyName));
    }
    
    protected bool SetProperty<T>(ref T field, T value, [CallerMemberName] string propertyName = null)
    {
        if (!EqualityComparer<T>.Default.Equals(field, value))
        {
            field = value;
            OnPropertyChanged(propertyName);
            return true;
        }
        return false;
    }
}
```

### MainViewModel

**职责**：主界面业务逻辑

```csharp
public class MainViewModel : ViewModelBase
{
    private ObservableCollection<ProxyRowViewModel> _proxies;
    private ProxyRowViewModel _selectedProxy;
    private string _computerName;
    private string _localIP;
    private string _networkStatus;
    private DateTime _currentTime;
    private ObservableCollection<string> _logs;
    
    // 属性（绑定到 View）
    public ObservableCollection<ProxyRowViewModel> Proxies
    {
        get => _proxies;
        set => SetProperty(ref _proxies, value);
    }
    
    public ProxyRowViewModel SelectedProxy
    {
        get => _selectedProxy;
        set => SetProperty(ref _selectedProxy, value);
    }
    
    public string ComputerName
    {
        get => _computerName;
        set => SetProperty(ref _computerName, value);
    }
    
    public string LocalIP
    {
        get => _localIP;
        set => SetProperty(ref _localIP, value);
    }
    
    public string NetworkStatus
    {
        get => _networkStatus;
        set => SetProperty(ref _networkStatus, value);
    }
    
    public DateTime CurrentTime
    {
        get => _currentTime;
        set => SetProperty(ref _currentTime, value);
    }
    
    public ObservableCollection<string> Logs
    {
        get => _logs;
        set => SetProperty(ref _logs, value);
    }
    
    // 命令
    public ICommand AddCommand { get; }
    public ICommand DeleteCommand { get; }
    public ICommand TestCommand { get; }
    public ICommand StartCommand { get; }
    public ICommand StopCommand { get; }
    
    public MainViewModel()
    {
        // 初始化集合
        Proxies = new ObservableCollection<ProxyRowViewModel>();
        Logs = new ObservableCollection<string>();
        
        // 初始化命令
        AddCommand = new RelayCommand(OnAddProxyClick);
        DeleteCommand = new RelayCommand(OnDeleteProxyClick, () => SelectedProxy != null);
        TestCommand = new RelayCommand(OnTestProxyClick, () => SelectedProxy != null);
        StartCommand = new RelayCommand(OnStartProxyClick, () => SelectedProxy != null);
        StopCommand = new RelayCommand(OnStopProxyClick, () => SelectedProxy != null);
        
        // 启动定时器刷新时间
        StartTimers();
        
        // 加载初始数据
        LoadProxies();
    }
    
    // 命令处理方法
    private async void OnAddProxyClick()
    {
        var window = new AddProxyWindow();
        window.ShowDialog();
        
        // 刷新列表
        await LoadProxies();
    }
    
    private async void OnTestProxyClick()
    {
        var result = await ProxyTester.TestAsync(SelectedProxy.ProxyItem);
        AddLog($"测试结果: {(result.Success ? "成功" : "失败")} (延迟: {result.DelayMs}ms)");
    }
    
    private async void OnStartProxyClick()
    {
        bool success = await ProxyProcessManager.StartAsync(SelectedProxy.ProxyItem);
        if (success)
        {
            SelectedProxy.Status = "Running";
            AddLog($"代理 {SelectedProxy.Name} 已启动");
        }
    }
    
    private async void OnStopProxyClick()
    {
        bool success = await ProxyProcessManager.StopAsync(SelectedProxy.ProxyItem);
        if (success)
        {
            SelectedProxy.Status = "Stopped";
            AddLog($"代理 {SelectedProxy.Name} 已停止");
        }
    }
    
    private void AddLog(string message)
    {
        Logs.Add($"[{DateTime.Now:HH:mm:ss}] {message}");
    }
    
    private void StartTimers()
    {
        // 每 1 秒更新时间
        var timer1 = new DispatcherTimer { Interval = TimeSpan.FromSeconds(1) };
        timer1.Tick += (s, e) => CurrentTime = DateTime.Now;
        timer1.Start();
        
        // 每 2 秒检测网络状态
        var timer2 = new DispatcherTimer { Interval = TimeSpan.FromSeconds(2) };
        timer2.Tick += (s, e) => RefreshNetworkStatus();
        timer2.Start();
    }
}
```

### AddProxyViewModel

**职责**：添加代理弹窗的业务逻辑

```csharp
public class AddProxyViewModel : ViewModelBase
{
    private string _proxyName;
    private string _remoteHost;
    private string _remotePort;
    private string _username;
    private string _password;
    private string _errorMessage;
    
    public string ProxyName { ... }
    public string RemoteHost { ... }
    public string RemotePort { ... }
    public string Username { ... }
    public string Password { ... }
    
    public ICommand TestCommand { get; }
    public ICommand SaveCommand { get; }
    public ICommand CancelCommand { get; }
    
    public AddProxyViewModel()
    {
        TestCommand = new RelayCommand(OnTestClick);
        SaveCommand = new RelayCommand(OnSaveClick);
        CancelCommand = new RelayCommand(OnCancelClick);
    }
    
    private async void OnTestClick()
    {
        // 验证输入
        if (!ValidateInput()) return;
        
        // 构造代理对象
        var proxyItem = new ProxyItem
        {
            RemoteHost = RemoteHost,
            RemotePort = int.Parse(RemotePort),
            Username = Username,
            Password = Password
        };
        
        // 测试连通性
        var result = await ProxyTester.TestAsync(proxyItem);
        
        if (result.Success)
        {
            ErrorMessage = $"测试成功，延迟: {result.DelayMs}ms";
        }
        else
        {
            ErrorMessage = $"测试失败: {result.ErrorMessage}";
        }
    }
    
    private async void OnSaveClick()
    {
        if (!ValidateInput()) return;
        
        var proxyItem = new ProxyItem
        {
            Name = ProxyName,
            RemoteHost = RemoteHost,
            RemotePort = int.Parse(RemotePort),
            Username = Username,
            Password = Password
        };
        
        // 分配本地端口
        int localPort = PortManager.AllocatePort();
        proxyItem.LocalPort = localPort;
        proxyItem.LocalHost = "127.0.0.1";
        
        // 保存到配置
        await ProxyDataService.AddProxyAsync(proxyItem);
        
        // 关闭窗口
        OnCancelClick();
    }
    
    private bool ValidateInput()
    {
        if (string.IsNullOrEmpty(ProxyName)) { ErrorMessage = "代理名称不能为空"; return false; }
        if (string.IsNullOrEmpty(RemoteHost)) { ErrorMessage = "代理 IP 不能为空"; return false; }
        if (!int.TryParse(RemotePort, out var port) || port < 1 || port > 65535)
        { ErrorMessage = "代理端口范围应在 1-65535"; return false; }
        if (string.IsNullOrEmpty(Username)) { ErrorMessage = "用户名不能为空"; return false; }
        if (string.IsNullOrEmpty(Password)) { ErrorMessage = "密码不能为空"; return false; }
        
        return true;
    }
}
```

## Model 层

### ProxyRowViewModel

**职责**：DataGrid 中单行代理的 ViewModel

```csharp
public class ProxyRowViewModel : ViewModelBase
{
    private ProxyItem _proxyItem;
    
    public ProxyItem ProxyItem => _proxyItem;
    
    public string Name
    {
        get => _proxyItem.Name;
        set { _proxyItem.Name = value; OnPropertyChanged(); }
    }
    
    public string RemoteHost => _proxyItem.RemoteHost;
    public int RemotePort => _proxyItem.RemotePort;
    public int LocalPort => _proxyItem.LocalPort;
    
    public string Status
    {
        get => _proxyItem.Status.ToString();
        set
        {
            if (Enum.TryParse<ProxyStatus>(value, out var status))
            {
                _proxyItem.Status = status;
                OnPropertyChanged();
            }
        }
    }
    
    public ProxyRowViewModel(ProxyItem proxyItem)
    {
        _proxyItem = proxyItem;
    }
}
```

## 数据绑定

### 绑定类型

| 绑定类型 | 说明 | 示例 |
|---|---|---|
| **OneWay** | 数据单向流向（Model → View） | `<TextBlock Text="{Binding Name}" />` |
| **TwoWay** | 数据双向流动 | `<TextBox Text="{Binding Name, Mode=TwoWay}" />` |
| **OneWayToSource** | 数据反向流向（View → Model） | 少用 |
| **OneTime** | 绑定一次，之后不更新 | 性能优化 |

### 更新触发

| 设置 | 说明 |
|---|---|
| **UpdateSourceTrigger=PropertyChanged** | 每次属性改变立即更新 |
| **UpdateSourceTrigger=LostFocus** | 控件失焦时更新 |
| **UpdateSourceTrigger=Explicit** | 手动调用更新 |

## 命令系统

### RelayCommand（命令实现）

```csharp
public class RelayCommand : ICommand
{
    private readonly Action _execute;
    private readonly Func<bool> _canExecute;
    
    public event EventHandler CanExecuteChanged;
    
    public RelayCommand(Action execute, Func<bool> canExecute = null)
    {
        _execute = execute;
        _canExecute = canExecute ?? (() => true);
    }
    
    public bool CanExecute(object parameter)
    {
        return _canExecute();
    }
    
    public void Execute(object parameter)
    {
        _execute();
    }
    
    public void RaiseCanExecuteChanged()
    {
        CanExecuteChanged?.Invoke(this, EventArgs.Empty);
    }
}

public class RelayCommand<T> : ICommand
{
    // 泛型版本，支持参数传递
}
```

## 最佳实践

### 1. 避免代码后置（Code Behind）

❌ **不好**：在 MainWindow.xaml.cs 中写业务逻辑

✅ **好**：将逻辑写在 ViewModel 中，XAML 绑定过来

### 2. 属性更新通知

❌ **不好**：
```csharp
public string Name { get; set; }
```

✅ **好**：
```csharp
private string _name;
public string Name
{
    get => _name;
    set => SetProperty(ref _name, value);
}
```

### 3. ObservableCollection 用于列表

✅ 集合项变更会自动通知 UI 更新

```csharp
Proxies = new ObservableCollection<ProxyRowViewModel>();
Proxies.Add(new ProxyRowViewModel(...));  // UI 自动刷新
```

### 4. 异步操作

✅ 长时操作使用 async/await，避免 UI 卡顿

```csharp
private async void OnTestProxyClick()
{
    var result = await ProxyTester.TestAsync(...);
}
```
