### 版本阶段

| 阶段 | 版本前缀 | 说明 |
|---|---|---|
| 开发中 | v0.x.x | 功能逐步完善，API/配置可能变更 |

## V1.0 完成标准

以下标准全部达成后，标记为 v1.0.0 稳定版：

### 基础运维

- [x] 双击 EXE 直接打开，零配置启动
- [ ] 安装包制作完成，支持一键安装
- [ ] 支持开机自启选项

### 网络隔离（核心约束）

- [x] 不修改 Windows 系统代理设置
- [x] 不修改网络适配器、系统路由、DNS
- [x] 不影响浏览器、微信、游戏等其他程序网络
- [x] 只有主动填写本地端口的软件经过代理

### 代理功能

- [x] 输入 HTTP 代理 IP + 端口 + 用户名 + 密码
- [x] 自动生成本地无认证端口
- [x] 每个本地端口独立对应一个上游代理
- [x] 本地端口范围：9001~9100
- [x] 支持代理连通性测试（含延迟测量）
- [x] 支持独立启动 / 停止单个代理

### 监控与状态

- [x] 后台 5 秒轮询，自动检测代理进程状态
- [x] 进程异常退出自动标记为 Failed
- [x] GUI 实时显示代理状态（Running / Stopped / Failed）

### 数据持久化

- [x] 代理配置保存到 config.json，重启不丢失
- [ ] 账号密码字段通过 DPAPI 加密存储（Phase 8）

### 图形界面

- [x] 显示本机域名、IP、网络状态、当前时间
- [x] 代理列表 DataGrid 展示完整信息
- [x] 添加代理弹窗（含表单校验）
- [x] 运行日志实时输出

## V1.0 → V2.0 规划

### V1.0.x（维护期）

- 缺陷修复
- 兼容性调整
- 文档完善

### V2.0.0 功能预览

| 功能 | 说明 |
|---|---|
| REST API 端点层 | 为外部工具提供 HTTP 接口控制代理 |
| SQLite 数据库 | 替代 JSON 文件存储，提升可靠性 |
| DPAPI 加密 | 密码字段加密存储 |
| 暗色主题 | 适应不同使用场景 |
| 系统托盘 | 最小化到托盘持续运行 |
| 批量操作 | 批量导入/导出、全部启动/停止 |
| 流量统计 | 连接记录、流量数据 |
| 自动更新 | 在应用内检查并安装更新 |
