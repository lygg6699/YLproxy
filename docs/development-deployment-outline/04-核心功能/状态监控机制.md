## 监控架构

### 后台监控流程

```
┌──────────────────┐
│  应用启动 (Main) │
└────────┬─────────┘
         │
         ▼
┌────────────────────────┐
│ MonitorService.Start() │  ← 后台启动定时器
└────────┬───────────────┘
         │
         ▼
    ┌─────────┐
    │ Timer   │  5 秒间隔
    │ Tick    │  ─────────┐
    └────┬────┘           │
         │                │
         ▼                │
  ┌──────────────────┐    │
  │ CheckStatusAsync │◄───┘
  └────────┬─────────┘
           │
           ├─ 遍历所有 Running 代理
           ├─ 调用 IsRunning()
           ├─ 检查进程存活
           │
           ▼
      ┌────────────┐
      │ 进程死亡？  │
      └────┬───┬───┘
           │   └─ 是 → 标记为 Failed → 通知 UI
           └─ 否 → 保持 Running
```

### 关键参数

| 参数 | 值 | 说明 |
|---|---|---|
| **检测间隔** | 5 秒 | 每 5 秒执行一次状态检测 |
| **超时时间** | 10 秒 | 超过 10 秒无响应判定为异常 |
| **重试次数** | 3 次 | 异常后重试 3 次（Phase 8+） |

## MonitorService 实现

### 核心代码

```csharp
public class MonitorService : IDisposable
{
    private readonly IProxyProcessManager _processManager;
    private readonly List<ProxyItem> _proxies;
    private Timer _timer;
    private bool _isMonitoring;
    
    // 事件通知 UI
    public event EventHandler<ProxyStatusChangedEventArgs> StatusChanged;
    
    public MonitorService(IProxyProcessManager processManager, List<ProxyItem> proxies)
    {
        _processManager = processManager;
        _proxies = proxies;
    }
    
    /// <summary>
    /// 启动监控服务
    /// </summary>
    public void Start()
    {
        if (_isMonitoring) return;
        
        _isMonitoring = true;
        
        // 创建 Timer，5 秒执行一次
        _timer = new Timer(
            state: null,
            dueTime: TimeSpan.FromSeconds(5),
            period: TimeSpan.FromSeconds(5),
            callback: async (obj) =>
            {
                try
                {
                    await CheckStatusAsync();
                }
                catch (Exception ex)
                {
                    Logger.Error($"监控服务异常: {ex.Message}");
                }
            }
        );
        
        Logger.Info("监控服务已启动");
    }
    
    /// <summary>
    /// 停止监控服务
    /// </summary>
    public void Stop()
    {
        _isMonitoring = false;
        _timer?.Dispose();
        Logger.Info("监控服务已停止");
    }
    
    /// <summary>
    /// 检查所有代理状态（5 秒执行一次）
    /// </summary>
    private async Task CheckStatusAsync()
    {
        if (!_isMonitoring) return;
        
        foreach (var proxy in _proxies)
        {
            // 只检查 Running 状态的代理
            if (proxy.Status != ProxyStatus.Running)
            {
                continue;
            }
            
            // 检查进程是否还活着
            bool isRunning = _processManager.IsRunning(proxy);
            
            if (!isRunning)
            {
                // 进程异常退出
                proxy.Status = ProxyStatus.Failed;
                
                Logger.Warning($"代理 '{proxy.Name}' 的进程异常退出");
                
                // 通知 UI 更新
                StatusChanged?.Invoke(this, new ProxyStatusChangedEventArgs
                {
                    ProxyId = proxy.Id,
                    OldStatus = ProxyStatus.Running,
                    NewStatus = ProxyStatus.Failed
                });
            }
        }
    }
    
    public void Dispose()
    {
        Stop();
        _timer?.Dispose();
    }
}
```

### 使用示例

```csharp
// 在 MainWindow 中初始化
public MainWindow()
{
    InitializeComponent();
    
    // 创建并启动监控服务
    var monitor = new MonitorService(_processManager, _proxies);
    monitor.StatusChanged += OnProxyStatusChanged;
    monitor.Start();
}

// 监听状态变化事件
private void OnProxyStatusChanged(object sender, ProxyStatusChangedEventArgs e)
{
    // 更新 UI
    var proxy = _proxies.FirstOrDefault(p => p.Id == e.ProxyId);
    if (proxy != null)
    {
        proxy.Status = e.NewStatus;
        RefreshUI();
    }
}
```

## 状态流转

### 完整状态机

```
┌─────────┐
│ Created │  (新建)
└────┬────┘
     │ StartAsync()
     ▼
┌─────────────┐
│  Starting   │  (启动中)
└────┬────────┘
     │
     ├─ 启动成功 → Running
     └─ 启动失败 → Failed
     
Running ◄─────────────────┐
  │                       │
  ├─ StopAsync()          │
  │   → Stopping          │
  │   → Stopped           │
  │                       │
  ├─ 5秒监控发现进程死亡  │
  │   → Failed            │
  │                       │
  └─ (正常运行中)◄────────┘
      (自动恢复，Phase 8+)

Failed
  │
  ├─ 用户点击"启动"
  │   → Starting
  │   → Running
  │
  └─ (等待用户恢复)
```

### 状态定义

| 状态 | 说明 | 进程 | 转移触发 |
|---|---|---|---|
| **Created** | 新建，未启动 | 无 | 用户点击"启动" |
| **Starting** | 启动中 | 创建中 | 进程创建完成或失败 |
| **Running** | 运行中，可转发 | 活跃 | 监控发现进程死亡或用户停止 |
| **Stopping** | 停止中 | 终止中 | 进程终止完成 |
| **Stopped** | 已停止 | 无 | 用户点击"启动"或应用启动 |
| **Failed** | 异常 | 无/已死亡 | 用户手动恢复或自动恢复(Phase 8+) |

## 故障检测

### 检测机制

```csharp
public bool IsRunning(ProxyItem proxy)
{
    try
    {
        // 1. 检查进程是否在追踪中
        if (!_processes.TryGetValue(proxy.Id, out var process))
        {
            return false;  // 未在追踪中，判定为不运行
        }
        
        // 2. 检查进程是否已退出
        if (process.HasExited)
        {
            _processes.Remove(proxy.Id);
            return false;  // 进程已退出
        }
        
        // 3. 额外验证：检查监听端口
        if (!IsPortListening(proxy.LocalPort))
        {
            return false;  // 端口未被监听
        }
        
        return true;  // 进程正常运行
    }
    catch (Exception ex)
    {
        Logger.Error($"检查进程状态异常: {ex.Message}");
        return false;  // 异常时假设进程不运行
    }
}

private bool IsPortListening(int port)
{
    try
    {
        var ipGlobalProperties = IPGlobalProperties.GetIPGlobalProperties();
        var tcpListeners = ipGlobalProperties.GetActiveTcpListeners();
        
        return tcpListeners.Any(endpoint => endpoint.Port == port);
    }
    catch
    {
        return false;
    }
}
```

## 自动恢复

### Phase 8 规划

```csharp
public class AutoRecoveryService
{
    public async Task RecoverFailedProxiesAsync()
    {
        foreach (var proxy in _proxies.Where(p => p.Status == ProxyStatus.Failed))
        {
            Logger.Info($"尝试自动恢复代理: {proxy.Name}");
            
            // 等待 30 秒后重试
            await Task.Delay(TimeSpan.FromSeconds(30));
            
            // 尝试重启
            bool success = await _processManager.StartAsync(proxy);
            
            if (success)
            {
                proxy.Status = ProxyStatus.Running;
                Logger.Info($"代理 {proxy.Name} 自动恢复成功");
                
                // 发送通知
                SendNotification($"代理 {proxy.Name} 已自动恢复");
            }
            else
            {
                Logger.Error($"代理 {proxy.Name} 自动恢复失败");
            }
        }
    }
}
```
