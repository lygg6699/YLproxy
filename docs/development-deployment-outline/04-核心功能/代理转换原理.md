## HTTP 代理协议

### 基本概念

HTTP 代理是一个中间服务，代表客户端向上游代理或目标服务器发送请求。

#### 工作流程

```
客户端              本地代理 (9001)      上游代理              目标网站
  │                    │                  │                  │
  ├─ HTTP 请求 ───────→│                  │                  │
  │                    │                  │                  │
  │                    ├─ 转发请求 ───────→│                  │
  │                    │                  │                  │
  │                    │                  ├─ 转发请求 ───────→│
  │                    │                  │                  │
  │                    │                  │◄─── 响应 ────────┤
  │                    │◄─ 转发响应 ───────│                  │
  │                    │                  │                  │
  │◄─ HTTP 响应 ───────│                  │                  │
  │                    │                  │                  │
```

### CONNECT 方法

对于 HTTPS 请求，使用 CONNECT 建立隧道：

```
CONNECT example.com:443 HTTP/1.1
Host: example.com
Proxy-Connection: keep-alive
```

## 认证机制

### Basic Authentication（基本认证）

#### 原理

1. 服务器要求认证
2. 客户端在请求头中加入 `Proxy-Authorization`
3. 格式：`Basic base64(username:password)`

#### 示例

```
原始认证信息：
  Username: test_user
  Password: password

Base64 编码：
  dGVzdF91c2VyOnBhc3N3b3Jk

HTTP 请求头：
  Proxy-Authorization: Basic dGVzdF91c2VyOnBhc3N3b3Jk
```

#### 实现

```csharp
using System.Net;
using System.Net.Http;

public async Task<HttpResponseMessage> SendRequestThroughProxyAsync(
    string targetUrl,
    string proxyHost,
    int proxyPort,
    string username,
    string password)
{
    // 1. 创建 WebProxy
    var proxy = new WebProxy(
        $"http://{proxyHost}:{proxyPort}",
        false,
        null,
        new NetworkCredential(username, password)
    );
    
    // 2. 创建 HttpClientHandler
    var handler = new HttpClientHandler
    {
        Proxy = proxy,
        UseProxy = true
    };
    
    // 3. 创建 HttpClient
    using (var client = new HttpClient(handler))
    {
        // 4. 发送请求
        var response = await client.GetAsync(targetUrl);
        return response;
    }
}
```

## 请求转换流程

### 完整转换过程

```
用户应用（模拟器等）
      │
      ▼
连接 127.0.0.1:9001（本地代理）
      │
      ▼
本地代理（3proxy）接收连接
      │
      ├─ HTTP 请求
      │    GET http://example.com/path HTTP/1.1
      │    Host: example.com
      │
      ▼
解析请求，识别目标主机
      │
      ├─ 目标：example.com
      ├─ 端口：80 (HTTP)
      │
      ▼
根据配置连接上游代理（123.123.123.123:8000）
      │
      ▼
上游代理要求认证
      │
      ├─ 407 Proxy Authentication Required
      │
      ▼
本地代理加入认证信息，重新发送
      │
      ├─ Proxy-Authorization: Basic [encoded_credentials]
      │
      ▼
上游代理验证通过，转发请求到目标网站
      │
      ├─ GET http://example.com/path HTTP/1.1
      │ (已剥离本地代理的代理信息)
      │
      ▼
目标网站响应
      │
      ▼
上游代理转发响应回本地代理
      │
      ▼
本地代理转发响应回用户应用
      │
      ▼
用户应用接收响应
```

## 实现细节

### ProxyTester 测试原理

```csharp
public class ProxyTester : IProxyTester
{
    public async Task<ProxyTestResult> TestAsync(ProxyItem proxy)
    {
        try
        {
            // 1. 创建 WebProxy 对象
            var webProxy = new WebProxy(
                $"http://{proxy.RemoteHost}:{proxy.RemotePort}",
                false,
                null,
                new NetworkCredential(proxy.Username, proxy.Password)
            );
            
            // 2. 创建 HttpClientHandler
            var handler = new HttpClientHandler
            {
                Proxy = webProxy,
                UseProxy = true,
                // 关闭证书验证（测试用，生产环境应启用）
                ServerCertificateCustomValidationCallback = (msg, cert, chain, errors) => true
            };
            
            // 3. 创建 HttpClient（设置超时）
            using (var client = new HttpClient(handler, disposeHandler: true))
            {
                client.Timeout = TimeSpan.FromSeconds(10);
                
                // 4. 准备测试请求
                var request = new HttpRequestMessage(
                    HttpMethod.Get,
                    "http://www.baidu.com"  // ProxyTester.cs 的实际测试端点
                );
                
                // 5. 测量耗时
                var stopwatch = Stopwatch.StartNew();
                
                // 6. 发送请求
                var response = await client.SendAsync(request);
                
                stopwatch.Stop();
                
                // 7. 判定结果
                if (response.IsSuccessStatusCode)
                {
                    return new ProxyTestResult
                    {
                        Success = true,
                        DelayMs = (int)stopwatch.ElapsedMilliseconds,
                        ErrorMessage = null
                    };
                }
                else if (response.StatusCode == System.Net.HttpStatusCode.ProxyAuthenticationRequired)
                {
                    return new ProxyTestResult
                    {
                        Success = false,
                        DelayMs = 0,
                        ErrorMessage = "认证失败：用户名或密码错误"
                    };
                }
                else
                {
                    return new ProxyTestResult
                    {
                        Success = false,
                        DelayMs = 0,
                        ErrorMessage = $"HTTP {response.StatusCode}: {response.ReasonPhrase}"
                    };
                }
            }
        }
        catch (HttpRequestException ex)
        {
            return new ProxyTestResult
            {
                Success = false,
                DelayMs = 0,
                ErrorMessage = $"连接失败：{ex.Message}"
            };
        }
        catch (OperationCanceledException ex)
        {
            return new ProxyTestResult
            {
                Success = false,
                DelayMs = 0,
                ErrorMessage = "超时：代理响应时间超过 10 秒"
            };
        }
        catch (Exception ex)
        {
            return new ProxyTestResult
            {
                Success = false,
                DelayMs = 0,
                ErrorMessage = $"异常：{ex.GetType().Name} - {ex.Message}"
            };
        }
    }
}
```

### 代理链的特殊处理

#### 场景：多级代理

```
应用 → YLproxy (9001) → 上游代理 1 (8080) → 上游代理 2 (9999) → 目标网站

当前实现：支持 1 级（应用 → 上游代理）
未来规划 (Phase XX)：支持多级链式代理
```
