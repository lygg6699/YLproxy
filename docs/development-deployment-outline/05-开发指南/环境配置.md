## 系统需求

### 最低要求

| 组件 | 版本 | 说明 |
|---|---|---|
| **操作系统** | Windows 10 / 11 | 仅支持 Windows |
| **.NET SDK** | 10.0+ | 从 global.json 可查看确切要求 |
| **Visual Studio** | 2022+ (可选) | 推荐，但可用 VS Code |
| **内存** | 4GB+ | 构建和运行时 |

### 检查系统要求

```powershell
# 查看 Windows 版本
Get-WmiObject win32_operatingsystem | Select-Object caption

# 查看 .NET SDK 要求
Get-Content global.json | ConvertFrom-Json | Select-Object sdk
```

## .NET SDK 安装

### 方式 1：从官方网站

1. 访问 https://dotnet.microsoft.com/download
2. 选择".NET 10" (或 global.json 中指定的版本)
3. 下载 Windows 安装程序
4. 运行安装程序，默认安装到 `C:\Program Files\dotnet`

### 方式 2：使用 VS Code

1. 安装 **C# Dev Kit** 扩展
2. 打开 YLproxy 文件夹
3. 提示缺失 SDK 时，点击"Install"
4. 自动下载安装（推荐）

### 方式 3：使用 Visual Studio Installer

1. 打开 **Visual Studio Installer**
2. 选择"修改"现有安装或"新建"安装
3. 勾选".NET Desktop Development"工作负载
4. 勾选".NET 10 SDK"
5. 点击"修改"安装

### 验证安装

```powershell
# 查看已安装的 .NET 版本
dotnet --list-sdks

# 输出示例：
# 10.0.100 [C:\Program Files\dotnet\sdk]

# 检查默认版本
dotnet --version
```

#### 必需扩展


```json

            "type": "coreclr",
            "request": "launch",

            "args": [],
            "cwd": "${workspaceFolder}",
            "stopAtEntry": false,
            "console": "internalConsole"
        }
    ]
}
```

### NuGet 包管理

#### 查看依赖

```powershell
# 列出所有 NuGet 包
dotnet list package

# 检查过期包
dotnet list package --outdated
```

#### 更新包

```powershell
# 更新特定包
dotnet package update <PackageName>

# 更新所有包
dotnet list package
```

### 构建步骤

#### 1. 清理

```powershell
# 清理所有构建产物
dotnet clean
```

#### 2. 还原

```powershell
# 还原 NuGet 包
dotnet restore
```

#### 3. 构建

```powershell
# 调试版本
dotnet build --configuration Debug

# 发布版本
dotnet build --configuration Release
```

#### 4. 运行

```powershell
# 运行 GUI 程序
dotnet run --project src/YLproxy.GUI/YLproxy.GUI.csproj

# 运行测试
dotnet test
```

## 常见问题

### Q1：.NET SDK 版本不匹配

**错误**：
```
error : A compatible version of the .NET SDK could not be found
```

**解决**：
```powershell
# 查看要求的版本
Get-Content global.json | ConvertFrom-Json

# 安装对应版本
# 从 https://dotnet.microsoft.com/download 下载
```

### Q2：NuGet 包无法下载

**错误**：
```
error NU1101: Unable to find package
```

**解决**：
```powershell
# 清理 NuGet 缓存
dotnet nuget locals all --clear

# 重新还原
dotnet restore --no-cache
```

### Q3：VS Code 无法识别项目

**解决**：
1. 安装 C# Dev Kit
2. 关闭并重新打开 VS Code
3. 按 F5 开始调试（会自动初始化）

### Q4：编译出现 XAML 错误

**错误**：
```
error MC3050: Cannot find the type 'MainWindow'
```

**解决**：
```powershell
# 清理并重新构建
dotnet clean
dotnet build
```
