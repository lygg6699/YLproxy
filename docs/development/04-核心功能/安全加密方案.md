## 当前安全状态

### 2026-07-15 已实现状态

- ✅ `DpapiSecurityService` 使用 Windows DPAPI `DataProtectionScope.CurrentUser`。
- ✅ `data/config.json` 中的用户名和密码使用 `dpapi:v1:` 前缀密文保存。
- ✅ `ProxyDataSerializer` 加载时解密旧格式并自动迁移，保存时始终加密。
- ✅ 配置写入使用临时文件和原子替换；迁移脚本失败时恢复原文件。
- ✅ 3proxy 运行 cfg 仅在进程生命周期内保留，停止或异常退出后删除。
- ⚠️ 密文绑定当前 Windows 用户，跨用户或跨计算机需要重新录入凭据。
- ⚠️ 运行中的 3proxy cfg 为引擎所需，短时间内包含明文上游凭据，必须由进程管理器清理.

### v0.2.0 历史风险（已处理）

#### ⚠️ 历史问题

在早期 v0.2.0 实现中，代理密码曾以**明文**存储在 config.json：

```json
{
  "proxies": [
    {
      "id": 1,
      "username": "test_user",
      "password": "test_password",  // ❌ 明文存储，风险！
      ...
    }
  ]
}
```

#### 风险场景

1. **文件暴露**：config.json 被他人获取 → 所有密码泄露
2. **备份泄露**：备份文件被他人访问 → 密码泄露
3. **内存泄露**：进程 dump 文件被分析 → 密码可能泄露
4. **日志泄露**：日志文件中可能记录密码

#### 影响范围

- 所有配置的代理账号密码都被泄露
- 攻击者可伪装成用户，使用受害者的代理

## DPAPI 加密原理

### 什么是 DPAPI？

**DPAPI** (Data Protection Application Programming Interface) 是 Windows 提供的数据加密机制。

#### 特点

- ✅ 基于 Windows 用户账号
- ✅ 自动密钥管理
- ✅ 无需手动保管密钥
- ✅ 与用户和计算机绑定

#### 加密过程

```
明文密码
    │
    ▼
DPAPI.ProtectedData.Protect()
    │
    ├─ 使用 Windows 用户密钥加密
    │
    ▼
密文 (Base64 编码)
    │
    ▼
保存到 config.json
```

#### 解密过程

```
密文 (Base64 编码)
    │
    ▼
Base64 解码
    │
    ▼
DPAPI.ProtectedData.Unprotect()
    │
    ├─ 自动检索用户密钥解密
    │
    ▼
明文密码 (仅在内存中)
    │
    ▼
使用于代理认证
```

## 实现方案

### DpapiSecurityService（Phase 8）

```csharp
using System.Security.Cryptography;

public interface ISecurityService
{
    string Encrypt(string plainText);
    string Decrypt(string cipherText);
}

public class DpapiSecurityService : ISecurityService
{
    private readonly DataProtectionScope _scope = DataProtectionScope.CurrentUser;
    
    /// <summary>
    /// 加密明文
    /// </summary>
    public string Encrypt(string plainText)
    {
        if (string.IsNullOrEmpty(plainText))
        {
            return null;
        }
        
        try
        {
            // 1. 转换为字节数组
            byte[] plainBytes = Encoding.UTF8.GetBytes(plainText);
            
            // 2. DPAPI 加密
            byte[] encryptedBytes = ProtectedData.Protect(
                plainBytes,
                null,  // 无额外 entropy
                _scope
            );
            
            // 3. 转换为 Base64 字符串
            string cipherText = Convert.ToBase64String(encryptedBytes);
            
            Logger.Debug($"密码已加密 (长度: {plainText.Length} → {cipherText.Length})");
            
            return cipherText;
        }
        catch (Exception ex)
        {
            Logger.Error($"加密失败: {ex.Message}");
            throw;
        }
    }
    
    /// <summary>
    /// 解密密文
    /// </summary>
    public string Decrypt(string cipherText)
    {
        if (string.IsNullOrEmpty(cipherText))
        {
            return null;
        }
        
        try
        {
            // 1. 转换为字节数组
            byte[] encryptedBytes = Convert.FromBase64String(cipherText);
            
            // 2. DPAPI 解密
            byte[] plainBytes = ProtectedData.Unprotect(
                encryptedBytes,
                null,  // 无额外 entropy
                _scope
            );
            
            // 3. 转换为字符串
            string plainText = Encoding.UTF8.GetString(plainBytes);
            
            Logger.Debug($"密码已解密");
            
            return plainText;
        }
        catch (CryptographicException ex)
        {
            Logger.Error($"解密失败 (可能是用户或计算机不同): {ex.Message}");
            throw;
        }
        catch (Exception ex)
        {
            Logger.Error($"解密异常: {ex.Message}");
            throw;
        }
    }
}
```

### 在 ProxyDataService 中使用

```csharp
public class ProxyDataService
{
    private readonly ISecurityService _securityService;
    
    /// <summary>
    /// 保存代理（密码自动加密）
    /// </summary>
    public async Task SaveProxiesAsync(List<ProxyItem> proxies)
    {
        // 1. 克隆列表，避免修改原数据
        var proxiesToSave = proxies.Select(p =>
        {
            return new ProxyItem
            {
                Id = p.Id,
                Name = p.Name,
                RemoteHost = p.RemoteHost,
                RemotePort = p.RemotePort,
                Username = p.Username,
                Password = _securityService.Encrypt(p.Password),  // ← 加密
                LocalHost = p.LocalHost,
                LocalPort = p.LocalPort,
                Status = p.Status,
                CreatedAt = p.CreatedAt,
                UpdatedAt = p.UpdatedAt
            };
        }).ToList();
        
        // 2. 序列化为 JSON
        var config = new { version = "1.0", proxies = proxiesToSave };
        var json = JsonSerializer.Serialize(config);
        
        // 3. 写入文件
        await File.WriteAllTextAsync(_configPath, json, Encoding.UTF8);
    }
    
    /// <summary>
    /// 加载代理（密码自动解密）
    /// </summary>
    public async Task<List<ProxyItem>> LoadProxiesAsync()
    {
        if (!File.Exists(_configPath))
        {
            return new List<ProxyItem>();
        }
        
        // 1. 读取文件
        var json = await File.ReadAllTextAsync(_configPath);
        var doc = JsonDocument.Parse(json);
        var proxiesArray = doc.RootElement.GetProperty("proxies");
        
        // 2. 反序列化
        var proxies = new List<ProxyItem>();
        foreach (var element in proxiesArray.EnumerateArray())
        {
            var proxy = JsonSerializer.Deserialize<ProxyItem>(element.GetRawText());
            
            // 3. 解密密码
            try
            {
                proxy.Password = _securityService.Decrypt(proxy.Password);  // ← 解密
            }
            catch (CryptographicException)
            {
                Logger.Warning($"代理 '{proxy.Name}' 的密码解密失败，跳过");
                proxy.Password = "";  // 清空密码
            }
            
            proxies.Add(proxy);
        }
        
        return proxies;
    }
}
```

## 迁移策略

### 从明文到加密的迁移（v0.2.0 → v0.3.0）

#### 迁移流程

```
v0.2.0 config.json （明文）
      │
      ▼
检测密码是否已加密（是否为 Base64）
      │
      ├─ 是 (已加密) → 跳过
      │
      ├─ 否 (明文) → 需要迁移
      │    │
      │    ▼
      │  加密所有密码
      │    │
      │    ▼
      │  保存为 v0.3.0 config.json （加密）
      │    │
      │    ▼
      │  备份原文件为 config.json.v0.2.0.bak
      │
      ▼
迁移完成
```

#### 实现

```csharp
public class MigrationService
{
    public async Task MigrateToEncryptionAsync()
    {
        // 1. 加载未加密的配置
        var proxies = await LoadPlainTextProxiesAsync();
        
        // 2. 检查是否需要迁移
        bool needsMigration = proxies.Any(p => !IsEncrypted(p.Password));
        
        if (!needsMigration)
        {
            Logger.Info("配置已是加密格式，无需迁移");
            return;
        }
        
        // 3. 备份原文件
        File.Copy("data/config.json", "data/config.json.v0.2.0.bak", overwrite: true);
        
        // 4. 加密所有密码
        foreach (var proxy in proxies)
        {
            if (!IsEncrypted(proxy.Password))
            {
                proxy.Password = _securityService.Encrypt(proxy.Password);
            }
        }
        
        // 5. 保存加密后的配置
        await _proxyDataService.SaveProxiesAsync(proxies);
        
        Logger.Info("密码加密迁移完成");
    }
    
    private bool IsEncrypted(string password)
    {
        // 简单判断：加密后的密码是 Base64 格式
        // 实际应使用更可靠的检测方法
        try
        {
            Convert.FromBase64String(password);
            return true;
        }
        catch
        {
            return false;
        }
    }
}
```

## 常见问题

### Q1：在另一台电脑上无法解密密码

**A：** DPAPI 与 Windows 用户账号绑定，不同电脑或用户无法解密。

**解决方案**：
- 使用 `DataProtectionScope.LocalMachine` 代替 `CurrentUser` (不推荐，安全性降低)
- 手动重新输入密码
- 使用可导出的加密方式（Phase 9+ 规划）

### Q2：忘记 Windows 密码，无法解密

**A：** 如果 Windows 用户被禁用或密码变更，配置无法解密。

**解决方案**：
1. 删除 config.json
2. 重新添加所有代理配置
3. 如有备份，恢复 config.json.v0.2.0.bak（但密码仍为明文，需重新加密）

### Q3：性能影响

**A：** 加密/解密性能不是瓶颈：
- 加密仅在保存时执行（低频）
- 解密仅在启动时执行（每次 1 次）
- 每个密码的加解密 < 1ms
