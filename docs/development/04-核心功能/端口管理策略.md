## 端口范围规划

### 为什么选择 9001-9100？

| 考虑因素 | 说明 |
|---|---|
| **不与系统冲突** | 9001-9100 是高端口，系统软件通常不占用 |
| **容量** | 100 个端口足以满足普通用户 |
| **易记忆** | 连续范围，便于识别 |
| **安全** | 无需管理员权限即可绑定 |

### 端口使用统计

```
总容量：100 个端口 (9001-9100)

预期用法：
- 普通用户：2-5 个代理
- 高级用户：10-20 个代理
- 企业用户：最多 100 个代理
```

## 分配算法

### 核心算法

```csharp
public class PortManager
{
    private const int MIN_PORT = 9001;
    private const int MAX_PORT = 9100;
    
    /// <summary>
    /// 分配一个可用的本地端口
    /// </summary>
    public int AllocatePort()
    {
        // 1. 遍历所有端口
        for (int port = MIN_PORT; port <= MAX_PORT; port++)
        {
            // 2. 检查是否在配置中已使用
            if (!IsPortInConfig(port))
            {
                // 3. 检查是否被系统占用
                if (!IsPortInUse(port))
                {
                    return port;  // ← 找到可用端口，返回
                }
            }
        }
        
        // 4. 无可用端口，抛异常
        throw new InvalidOperationException(
            $"端口耗尽：9001-9100 已全部占用，请删除不使用的代理后重试"
        );
    }
    
    /// <summary>
    /// 检查端口是否在配置中使用
    /// </summary>
    private bool IsPortInConfig(int port)
    {
        var proxies = _proxyDataService.LoadProxies();
        return proxies.Any(p => p.LocalPort == port);
    }
    
    /// <summary>
    /// 检查端口是否被系统占用
    /// </summary>
    private bool IsPortInUse(int port)
    {
        try
        {
            var ipGlobalProperties = IPGlobalProperties.GetIPGlobalProperties();
            var tcpListeners = ipGlobalProperties.GetActiveTcpListeners();
            
            return tcpListeners.Any(endpoint => endpoint.Port == port);
        }
        catch (Exception ex)
        {
            Logger.Warning($"无法检查端口状态: {ex.Message}");
            return false;  // 无法判断时假设端口未被占用
        }
    }
    
    /// <summary>
    /// 释放端口（用于删除代理时）
    /// </summary>
    public void ReleasePort(int port)
    {
        if (port < MIN_PORT || port > MAX_PORT)
        {
            throw new ArgumentOutOfRangeException(nameof(port));
        }
        
        Logger.Info($"端口 {port} 已释放，可被重新分配");
    }
}
```

### 分配流程

```
AddProxyWindow 弹窗
      │
      ▼
用户点击"保存"
      │
      ▼
[PortManager.AllocatePort()]
      │
      ├─ 遍历 9001
      │    ├─ 不在配置中 ✅
      │    ├─ 未被系统占用 ✅

      ▼

      │
      ▼

      │
      ▼

```

## 冲突检测

### 冲突类型

| 类型 | 说明 | 检测方法 |
|---|---|---|
| **配置冲突** | 两个代理配置了同一个本地端口 | 检查 config.json |
| **系统冲突** | 本地端口被其他程序占用 | 使用 IPGlobalProperties |
| **进程冲突** | 多个 3proxy 进程绑定同一端口 | 检查 Process PID |

### 冲突解决

```csharp
public class ConflictResolver
{
    /// <summary>
    /// 验证代理配置是否有冲突
    /// </summary>
    public List<ProxyConflict> DetectConflicts(List<ProxyItem> proxies)
    {
        var conflicts = new List<ProxyConflict>();
        
        // 1. 检查配置冲突（同一个端口被多个代理使用）
        var portGroups = proxies.GroupBy(p => p.LocalPort);
        foreach (var group in portGroups.Where(g => g.Count() > 1))
        {
            conflicts.Add(new ProxyConflict
            {
                Type = ConflictType.ConfigDuplicate,
                Port = group.Key,
                Proxies = group.Select(p => p.Name).ToList()
            });
        }
        
        // 2. 检查系统冲突（端口被其他程序占用）
        var ipGlobalProperties = IPGlobalProperties.GetIPGlobalProperties();
        var tcpListeners = ipGlobalProperties.GetActiveTcpListeners();
        var usedPorts = tcpListeners.Select(ep => ep.Port).ToHashSet();
        
        foreach (var proxy in proxies)
        {
            // 排除该代理自己的 3proxy 进程
            if (proxy.Status != ProxyStatus.Running && usedPorts.Contains(proxy.LocalPort))
            {
                conflicts.Add(new ProxyConflict
                {
                    Type = ConflictType.SystemOccupied,
                    Port = proxy.LocalPort,
                    Proxies = new List<string> { proxy.Name }
                });
            }
        }
        
        return conflicts;
    }
}
```

## 回收机制

### 删除代理时的端口释放

```csharp
public async Task DeleteProxyAsync(int proxyId)
{
    // 1. 获取代理对象
    var proxy = _proxies.FirstOrDefault(p => p.Id == proxyId);
    if (proxy == null) return;
    
    // 2. 如果代理正在运行，先停止
    if (proxy.Status == ProxyStatus.Running)
    {
        await _processManager.StopAsync(proxy);
    }
    
    // 3. 从配置中删除
    _proxies.Remove(proxy);
    await _dataService.SaveProxiesAsync(_proxies);
    
    // 4. 标记端口为已释放
    _portManager.ReleasePort(proxy.LocalPort);
    
    Logger.Info($"代理已删除，端口 {proxy.LocalPort} 释放");
}
```

### 异常恢复

如果程序崩溃，用户启动时自动回收孤儿进程占用的端口：

```csharp
public async Task RecoverFromCrash()
{
    // 1. 获取所有 3proxy 进程
    var processes = Process.GetProcessesByName("3proxy");
    
    // 2. 对于每个进程，检查是否在配置中存在
    foreach (var process in processes)
    {
        bool found = false;
        
        foreach (var proxy in _proxies)
        {
            if (proxy.ProcessId == process.Id)
            {
                found = true;
                break;
            }
        }
        
        // 3. 未在配置中的进程（孤儿），强制杀死
        if (!found)
        {
            try
            {
                process.Kill();
                Logger.Warning($"清理孤儿进程: PID {process.Id}");
            }
            catch (Exception ex)
            {
                Logger.Error($"清理孤儿进程失败: {ex.Message}");
            }
        }
    }
}
```

## 容量管理

### 容量监控

```csharp
public class CapacityMonitor
{
    public CapacityInfo GetCapacityInfo(List<ProxyItem> proxies)
    {
        var totalPorts = PortManager.MAX_PORT - PortManager.MIN_PORT + 1;
        var usedPorts = proxies.Count;
        var availablePorts = totalPorts - usedPorts;
        var utilizationRate = (usedPorts * 100.0) / totalPorts;
        
        return new CapacityInfo
        {
            TotalPorts = totalPorts,
            UsedPorts = usedPorts,
            AvailablePorts = availablePorts,
            UtilizationRate = utilizationRate,
            IsNearCapacity = utilizationRate > 80.0,
            IsFull = availablePorts == 0
        };
    }
    
    public void CheckAndWarn(CapacityInfo info)
    {
        if (info.IsFull)
        {
            Logger.Warning("⚠️ 警告：本地端口已全部占用，无法添加新代理");
        }
        else if (info.IsNearCapacity)
        {
            Logger.Warning(
                $"⚠️ 警告：端口利用率已达 {info.UtilizationRate:F1}%，" +
                $"还可添加 {info.AvailablePorts} 个代理"
            );
        }
    }
}
```

### 容量统计

| 配置数 | 利用率 | 状态 | 建议 |
|---|---|---|---|
| 0-10 | <10% | ✅ 充足 | 正常使用 |
| 10-80 | 10-80% | ✅ 良好 | 正常使用 |
| 80-95 | 80-95% | ⚠️ 紧张 | 考虑删除不用的代理 |
| 95-100 | 95-100% | ❌ 饱和 | 必须删除代理才能添加 |
