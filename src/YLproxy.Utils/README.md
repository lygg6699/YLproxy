# YLproxy.Utils/

## 目录简介

工具函数库，存放项目中通用的帮助函数和工具类，促进代码复用和保持项目结构清晰。

## 主要文件/子目录说明

### PathResolver.cs
统一解析仓库根目录和运行时路径：
- 优先从程序集、入口程序和当前目录向上查找有效的 `YLproxy.sln`
- 以仓库根目录为相对路径基准
- 发布目录没有解决方案文件时，回退到应用程序基目录
- 不把下级 `data`、`logs` 或其他运行目录误认为仓库根目录

## 使用说明

未来开发过程中，可以将以下真正通用的函数放置在此目录：
- 字符串处理工具（IP 地址验证、端口格式化等）
- 网络工具（DNS 查询、网络状态检测等）
- 文件操作工具（安全文件读写、目录操作等）
- 加密解密工具（DPAPI 包装等）
- 日志帮助类（格式化、过滤等）

## 注意事项

1. **保持通用性**：此目录中的函数应该是真正通用的，不应包含特定业务逻辑
2. **避免重复**：在添加新函数前，请检查是否已有类似功能的实现
3. **单元测试**：考虑为公共工具函数编写单元测试以确保可靠性
